# Introduction.

이 프로젝트는 프론트엔트 샘플 코드 저장소 입니다.

# Installation

프로젝트 다운로드 :

```
git clone --recursive https://github.com/nettee-space/frontend-sample-code-registry.git

```

프로젝트 다운로드 후 폴더가 비어 있을 시 :

```
# 다음 3가지 명령어를 실행 해주세요

cd frontend-sample-code-registry

git submodule init

git submodule update

```
